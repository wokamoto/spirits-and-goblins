== Description ==

このプラグインを有効にすることでワンタイムパスワードによる二段階認証を導入することができます。
ワンタイムパスワードはメールまたは、Twilio を使用して携帯電話へ Short Message として送信できます。

== Installation ==

1. Upload the entire `spirits-and-goblins` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

The control panel of Spirits and Goblins is in 'Settings > Spirits and Goblins'.
